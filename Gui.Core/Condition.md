# Condition Class

## namespace [Noesis](https://www.noesisengine.com/docs/Gui.Core._ClassHierarchy.html#noesis-namespace) | [MSDN](http://msdn.microsoft.com/en-us/library/system.windows.condition.aspx)

Represents a condition for the [MultiTrigger](https://www.noesisengine.com/docs/Gui.Core._MultiTrigger.html) and the [MultiDataTrigger](https://www.noesisengine.com/docs/Gui.Core._MultiDataTrigger.html), which apply changes to property values based on a set of conditions.
