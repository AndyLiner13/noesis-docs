# NoesisGUI Documentation

This repository contains converted documentation for NoesisGUI from HTML to Markdown format, organized in a folder structure similar to the hw-docs format.

## Contents

- **Animation/** - Animation system documentation
- **Binding/** - Data binding documentation  
- **Class Hierarchy/** - Class hierarchy documentation
- **Clocks/** - Animation clocks documentation
- **Dependency System/** - Dependency property system documentation
- **Easing Functions/** - Animation easing functions documentation
- **Element Tree Overview/** - Element tree documentation
- **Group Timelines/** - Group timeline animations documentation
- **Markup Extensions/** - XAML markup extensions documentation

## Original Source

These documents were originally from the NoesisGUI official documentation website and have been converted from HTML to Markdown format for easier reading and integration.

## NoesisGUI

NoesisGUI is a cross-platform XAML-based UI framework for games and real-time applications.

- Website: https://www.noesisengine.com/
- Documentation: https://www.noesisengine.com/docs/
